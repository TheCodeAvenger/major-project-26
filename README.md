# 🎓 EduTrack Pro

EduTrack Pro is a modern **School Student Management System** built using PHP and MySQL.  
It helps manage students, attendance, and timetable efficiently through a clean and interactive web interface.

---

## 🚀 Features

### 👥 Student Management
- Add, edit, delete students
- View all students in tabular format
- Search & filter by name, class, section

### 📝 Attendance System
- Mark attendance (Present / Absent)
- Date-wise attendance tracking
- One attendance per student per day

### 📅 Timetable Management
- Add timetable entries (class, section, subject, time)
- View timetable with filters
- Dashboard shows **today’s timetable**

### 📊 Dashboard
- Total students, classes, sections
- Attendance percentage
- Live insights with quick navigation

### 📈 Analytics
- Students per class
- Class & section-wise attendance report
- Structured data tables for better understanding

## ⚙️ Installation

1. Clone the repository :-
git clone https://github.com/TheCodeAvenger/edutrack-pro.git 

2.Move project to XAMPP htdocs folder

3.Start Apache & MySQL

4.Create database in phpMyAdmin :
edutrack

5.Import SQL tables

6.Run project : 
http://localhost/edutrack-pro/
